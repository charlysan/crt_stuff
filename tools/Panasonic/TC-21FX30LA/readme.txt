- ypbpr2rgb.ino

  Firmware for ESP8266 that will write to VSP to re-configure AV3 component input as RGB.

  Installation: 
    - Download esp-i2c library as .zip file and install it into Arduino IDE: https://github.com/juhovh/esp-i2c
    - Build and upload to ESP8266
  
  Tested with NodeMCU Mini.
